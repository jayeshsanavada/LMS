Zoho HR — Field Mapping
Probation gate relies on employee_status mapping. When Zoho sends employee_status = "Probation" or "New Joiner", the LMS sets employment_phase = PROBATION and auto-triggers gate training assignment.
Employee Phase Mapping
Zoho employee_status value LMS employment_phase Action triggered
"Probation" PROBATION Create user_probation record + auto-assign gate trainings
"New Joiner" PROBATION Create user_probation record + auto-assign gate trainings
"Active" CONFIRMED Normal user creation — standard training assignment rules apply
"Intern" INTERN User created — no probation gate triggered
"Resigned" / "Terminated" EXITED User deactivated — if in probation, probation_status = TERMINATED
